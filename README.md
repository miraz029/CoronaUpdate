# CoronaUpdate
A sample android app writen in Java that shows the latest CoronaVirues related news from free API's

### API
Free API is used from https://newsapi.org/docs to fetch CoronaVirus related news.

### Architecture
Used Android MVVM with Clean Architecture.

### Library Used
1. RxJava2
2. Dagger2
3. Room
4. Retrofit2
5. Okhttp3
6. Gson
7. Glide

### Packages:
1. data
2. di
3. domain
4. presentation
4. utils
